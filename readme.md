# Proyecto: Mostrar Usuario (Promesas y Async/Await)

Este proyecto es una práctica en **JavaScript**, **HTML** y **CSS**, orientada a comprender el funcionamiento de las **promesas** y del manejo **asíncrono** mediante `async` / `await`.

El objetivo es simular la obtención de datos de un usuario y mostrarlos dinámicamente en pantalla al hacer clic en un botón.

---

## Funcionalidad

- Botón que inicia la acción.
- Simulación de una operación asíncrona.
- Uso de `Promise` y `async / await` en el mismo script.
- Impresión dinámica en el DOM.
- Fondo con imagen y estilos gradientes.
